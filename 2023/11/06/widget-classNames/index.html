简介

  • 用于有条件地将类名连接在一起，常用于 React 应用。

基本使用

  • 该classNames函数接受任意数量的参数，可以是字符串或对象。该参数'foo'是 的缩写{ foo: true }。
  • 如果与给定键关联的值是假的，则该键将不会包含在输出中。
  • 配合模版字符串使用
  • 通过 classname/bind 实现类名合并。
  • 注：如果有重复的类名不会被去重
